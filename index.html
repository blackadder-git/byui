Sudoku Solver

CS124: Intro to Software Development

Structure Chart
C++ Cohesive Loose Coupling Robust Error Checking Elegant Design

Project Outcomes

  • The objective of this project was to create a sudoku solver with a menu that would allow a user to select options to see the menu, display the board, edit a square, exit. In addition, extra credit was offered if the program could solve the puzzle on its own. Once I had the game working, I looked into using backtracing to complete the game. While this would work, it took a while for the program to solve the board.
  • I decided to look into how a human being might approach a sudoku puzzle. I soon learned about techniques such as hidden singles, hidden pairs, and pointing pairs. I implemented these search techniques into my program with a temporary board and found that the puzzles were solved much quicker.
