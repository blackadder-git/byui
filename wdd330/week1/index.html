    • Localstorage
      • All pages, from one origin, can store and access the same data.
      • window.localStorage (no expiration) vs window.sessionStorage (expires with session)
      • To view and delete via Chrome, open the developers tools and click on the Application tab to find (Local Storage, Session Storage, Cookies, etc)
      • Local storage only supports strings, use JSON.stringify() and JSON.parse() to work with arrays
    • Github pages
      • Log in to Github and enter the repository
      • Goto settings
      • Click Pages under Options
      • Under Source, click main (master) in the dropdown
      • Make sure there is an index file in the directory
